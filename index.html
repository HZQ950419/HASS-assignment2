What is it?

This is a snippet of a full graphic, and is based on a recent peer reviewed article which analyzed “79 studies on the effects of stress and the human body”. This visualization shows how the work place stressors relate to physical symptoms.

Source https://visual.ly/community/Infographics/health/effects-work-stress-your-body

Issues

Data Issue:

  • There is no exact value in the visualization. The horizontal bar doesn't show the exact value of the correlation between work place stressors and physical symptoms.

Idiom Issue:

  • The horizontal bar is showing the volume of the occurrence of each symptom relative to the workplace stressor.
  • There is a relationship between the symptom and the stressor, but labeling the column header as Relationship is both confusing and misleading.
  • The column title should clearly state the unit of measure (e.g. count, sum of, % of, etc)

Possible Solutions

  • The column title can be Correlation and the column title should clearly state the unit of measure.
  • In this case the horizontal bar chart was the right choice, we can add the exact value to make it clear.

What is it?

This is a 3D model to show the correlation of the CpG island hypermethylation and human cancer. There are three variables, the percentage of methylation, the human organs, and the gene segments.

Issues

Idiom Issue:

  • The graph involves too many variables which makes it difficult for the viewer to comprehend it immediately.
  • The 3D design has resulted in undue complexity wherein some bars are simply seemed to be hidden behind those in the front
  • It is nearly impossible to distinguish the percentage of each bar.

Possible Solutions

  • Make the visualization interactive. If we can rotate the chart to view from different directions and click to check the exact value of each bar, the visualization can be better.
  • In this case, we can use 2D Charts with raw header Organ and column header Gene Segment, the value in table is the percentage of methylation. Moreover, we can use colors to indicate different extents of correlation.
